# UX Principles — Nexus

## 1. Design for Intent, Not Actions
Users express goals, not clicks. The system infers the interface from natural language — no forms, no menus, no friction.

## 2. Transparency Builds Trust
Every AI output shows: confidence score, reasoning summary, and sources. Nothing is hidden behind a "magic" black box.

## 3. Predict, Don't Assume
Suggestions appear contextually, never block the user. Predictive chips are dismissible with one click.

## 4. Fail Gracefully, Recover Beautifully
Errors are explained in plain language. Recovery paths are obvious. The system maintains context during failures.

## 5. Human-in-the-Loop
AI generates, human approves. Every output is editable inline. The user has final say, always.

## 6. Warmth Over Cold Tech
The interface feels like a premium notebook, not a server dashboard. Paper textures, ink accents, tactile feedback.

## 7. Accessible by Default
All interactions work with keyboard. Screen readers announce state changes. Motion respects user preferences.

## 8. Performance is Design
Fast transitions. Lazy loading. No jank. The interface should feel as responsive as a native app.
